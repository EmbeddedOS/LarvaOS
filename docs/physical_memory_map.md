# Linux kernel memory management

Physical Address Space

0x00000000  +---------------------------------------+
            | Interrupts vector table (1024 bytes). |
0x00000400  +---------------------------------------+
            | BIOS Data area (256 bytes).           |
0x00000500  +---------------------------------------+
            | OS load area (30,464 bytes).          |
0x00007C00  +---------------------------------------+
            | MBR (512 bytes).                      |
0x00007E00  +---------------------------------------+
            | Boot Stack/Data area (512 bytes).     |
0x00007FFF  +---------------------------------------+
            |                                       |
0x000A0000  +---------------------------------------+
            | Video memory.                         |
0x000C0000  +---------------------------------------+
            |                                       |
0x00010000  +---------------------------------------+
            | Kernel Image (.text, .data, .ro, etc.)|
            +---------------------------------------+
            | Free Low memory.                      |
0xC0000000  +---------------------------------------+
            | Free High memory.                     |
0xFFFFFFFF  +---------------------------------------+
